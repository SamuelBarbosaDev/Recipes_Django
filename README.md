### Ambiente virtual e Dependências:
Criando ambiente virtual:
```
python3.10 -m venv core/.venv
```

Entrando no ambiente virtual:
```
source core/.venv/bin/activate
```

Instale as dependências:
```
pip install -r core/requirements.txt
```
